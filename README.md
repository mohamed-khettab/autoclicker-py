# Python-Autoclicker

A Python Customizable Autoclicker. 

How to use:

Install the latest version of Python. Make sure you mark Add to PATH.

# If you're on Windows:

Open Command Prompt

pip install pynput

pip install pyautogui

Run the file.

# If you're on Mac/Linux:

Open Terminal

pip3 install pynput

pip3 install pyautogui

Run the file.
